# Mcurve

A Project to lern Python Module and PyPi

# Howto


* Setup an venv

Do this Task just onece after clone project

```
pip install venv
python -m venv .venv
```

Venv have to be activated (each times before development)

```
source .venv/bin/active
```

* Run the tests

Venv must be activated before

```
pip install -e .
```
